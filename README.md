# GHOST-CORE // Syahdan Alfaatih Portfolio

Personal portfolio website with a cinematic, cyberpunk, and terminal-inspired aesthetic. Built to showcase my experience in IT Infrastructure, System Operations, and Full-Stack Development.

## 💻 Tech Stack

- **Core:** React.js (v18)
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Animations:** Pure CSS (`@keyframes`) & Custom React Hooks
